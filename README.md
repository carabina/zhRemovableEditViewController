- Xcode 9.0+

## 安装
支持[CocoaPods](http://cocoapods.org)安装. 

```ruby
platform :ios, '7.0'
use_frameworks!

target 'You Project' do

pod 'zhRemovableEditViewController', '~> 0.1.0'

end
```
